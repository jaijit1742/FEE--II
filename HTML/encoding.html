3. URL Encoding Example

Sometimes URLs contain characters that need to be encoded.

Original URL:

https://example.com/search?query=html & css

Encoded URL:

https://example.com/search?query=html%20%26%20css

Some common encoded characters:

  • Space: %20
  • &: %26
  • =: %3D
  • ?: %3F
